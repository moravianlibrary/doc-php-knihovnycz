mixed getRelatedIdentities(string $name, int $maxRecords = 10)

Given a name string, get related identities. Inspired by Eric Lease Morgan's Name Finder demo (http://zoia.library.nd.edu/sandbox/name-finder/).

Return value is an associative array where key = author name and value = subjects used in that author's works.

Parameters

string $name

Name to search for (any format).
int $maxRecords

Max # of records to read from API (more = slower).

Return Value

mixed

False on error, otherwise array of related names.
