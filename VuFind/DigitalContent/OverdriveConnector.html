at line 374
object|bool getCollectionToken()

Get Colllection Token

Gets the colleciton token for the Overdrive collection. The collection token doesn't change much but according to the OD API docs it could change and should be retrieved each session. Also, the collection token depends on the user if the user is in a consortium. If consortium support is turned on then the user collection token will override the library collection token. The token itself is returned but it's also saved in the session and automatically returned.

Return Value

object|bool

A collection token for the library's collection.
